# RAM
ClassUn algoritmo es una serie de instrucciones que describen cómo realizar una tarea o resolver un problema. 

Aquí se orienta en su creación 

1. **Define el Problema**

Antes de escribir un algoritmo, debes entender el problema que deseas resolver. Considera lo siguiente:

- ¿Qué es lo que quieres lograr?
- ¿Cuáles son las entradas y las salidas del problema?

2. **Identifica las Entradas y Salidas**

- **Entradas**: Son los datos que necesita tu algoritmo para funcionar. Ejemplo: números, texto, listas.
- **Salidas**: Es el resultado que esperas obtener. Ejemplo: un número, un mensaje, una lista.

 3. **Estructura Básica de un Algoritmo**

Generalmente, un algoritmo se puede describir en forma de pasos o instrucciones. Aquí te muestro cómo se estructura:

1. **Inicio**: Marca el comienzo del algoritmo.

2. **Instrucciones**: Los pasos que se deben seguir.

3. **Condicionales**: Decisiones que cambian el flujo según ciertas condiciones.

4. **Repeticiones**: Pasos que se repiten varias veces (bucles).

5. **Fin**: Marca el final del algoritmo.
